
## Чекпоинт 1. Сбор данных

#### Как получаем данные

1. Собрали синтетический дотасет с помощью Qwen3 с вопросами по книге, релевантными фрагментами текста и эталонными ответами на них по 5 произведениям:
 * Мартин Иден, * Капитанская дочка,
* Герой нашего времени, * Отцы и дети, * Преступление и наказание. 
2. Синтетику отсмотрели глазами и сверили достоверность цитат с текстами произведений.

3. Собрали тексты книг в PDF. 

#### Формат данных

Книги — в формате PDF.

Эталонный датасет — в формате JSON:	
```json
[
    {
        "question": "Вопрос пользователя по книге",
        "relevant_text": "Фрагмент из книги, отвечающий на вопрос",
        "answer": "Ответ LLM на вопрос пользователя"
    },
...
]
```

#### Объем данных
* 5 книг (4 для экспериментов и 1 валидационная),
* 50 вопросов и ответов.

#### Обработка данных
1. Провели серию экспериментов по определению оптимальной конфигурации чанкинга (chunk_size, overlap, top-k). 
2. Оценили конфигурации с помощью DeepEval по следующим метрикам: answer_relevant, contextuall_recall, ans_faithfulness. Выбрали оптимальную конфигурацию (баланс top-k и качества).
3. Текст из PDF каждой книги нарезали на чанки и сложили в коллекции в Qdrant.
